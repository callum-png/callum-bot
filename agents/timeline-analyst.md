# X Timeline Trend Analyst

Research what's going viral RIGHT NOW on X/Twitter.

## Instructions

Option A (Chrome MCP available):
1. Navigate to https://x.com/home
2. Zoom out to 80% for more tweets visible
3. Scroll and capture 30-50 tweets with get_page_text
4. Note view counts, formats, hooks

Option B (Chrome unavailable -- fallback):
1. WebSearch for trending tweets in the relevant niche
2. Search for viral AI/tech content from last 48 hours
3. Search for top-performing formats this week

## Analyze Each Tweet For:
- View count and engagement rate
- Format: one-liner, thread, QRT, article, meme+caption, spectacle video
- Hook structure: first 10 words
- Emotional register: outrage, awe, humor, fear, aspiration, disbelief
- Which viral mechanism(s) used:
  M1=specific numbers, M2=emotional setup, M3=proxy audience, M4=impossible instruction
  M5=institutional failure, M6=zero-sum frame, M7=official source, M8=consequence closing

## Output:
1. TOP 5 performing formats right now
2. Which viral mechanisms are overperforming TODAY
3. What hooks are working this week
4. What emotional registers are hot vs saturated
5. Format recommendations for the current campaign
